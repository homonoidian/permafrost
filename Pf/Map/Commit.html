    class Pf::Map::Commit(K, V)

    • Pf::Map::Commit(K, V)
    • Reference
    • Object

    Overview

    Commits allow you to compose multiple edits into one, big edit of the map. Thus you avoid creating many useless intermediate copies of the map.

    Defined in:
